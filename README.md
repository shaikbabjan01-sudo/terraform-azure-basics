# Terraform Azure Basics

Terraform scripts for Azure infrastructure practice.

This repository contains my Terraform practice for Microsoft Azure infrastructure.

## Services Covered
- Virtual Machine
- Virtual Network (VNet)
- Subnet
- Network Security Group (NSG)
- Public IP

## Terraform Commands
- terraform init
- terraform plan
- terraform apply
- terraform destroy

## Goal
To practice Infrastructure as Code using Terraform on Azure.
